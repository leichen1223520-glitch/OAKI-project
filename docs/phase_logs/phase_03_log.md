## 改进说明（v1.0 → v2.0）

### 主要问题（v1.0）
1. 文献综述缺少具体数量数据（例：N₂O排放因子范围仅笼统提及，未给出Kampschreur等的原始统计）
2. 机理推导不完整（N₂O三条路径未分别推导动力学方程）
3. 参数选择方法论单薄（仅"打分"，缺少相关性矩阵和PCA的具体执行）
4. 子模型方程缺少关键推导步骤（如AOR中的4.33 gO₂/gN如何得出）
5. 贝叶斯率定代码不完整，缺少似然函数推导的正当性论证
6. 灵敏度分析结果是预期值而非分析框架的严格执行
7. 案例章节缺少逐月预测对比数据
8. 结论缺少与论文其他章节的数字呼应

   - 药剂LCA因子中国vs欧洲对比
   - AR4/AR5 GWP差异影响
4. **第三章**：
   - 四步筛选法完整执行（25→16→12→10，含表格）
