# 详细研究计划

## 研究题目
**基于AAO工艺污水处理厂轻量化数据下的全厂碳排放模型的构建研究**

---

## 一、研究背景与意义

### 1.1 政策背景
- "双碳"目标（2030年碳达峰、2060年碳中和）对污水处理行业提出减排要求
- 城镇污水处理厂是城市碳排放的重要来源（占城市排放约1-3%）
- 生态环境部推动污水处理行业碳排放核查标准化

### 1.2 技术背景
- AAO工艺是国内应用最广泛的脱氮除磷工艺，占比超过40%
- 现有全厂碳排放模型多依赖完整在线监测数据，实用性受限
- 轻量化数据驱动模型可降低应用门槛，具有重要工程价值

### 1.3 研究意义
- **理论意义**：完善污水处理厂碳排放核算方法论，填补轻量化数据条件下的研究空白
- **实践意义**：为中小型污水处理厂提供低成本、可操作的碳排放核算工具

## 三、技术方法

### 3.1 碳排放核算框架
采用IPCC第五次评估报告（AR5）100年全球增温潜势（GWP）系数：
- CO₂：GWP = 1
- CH₄：GWP = 28
- N₂O：GWP = 265

### 3.2 排放范围划分（参考GHG Protocol）
- **范围1（Scope 1）**：直接温室气体排放（CH₄、N₂O）
- **范围2（Scope 2）**：外购电力间接排放
- **范围3（Scope 3）**：药剂生产、污泥外运等上下游排放

### 3.3 轻量化数据定义
本研究中"轻量化数据"指污水处理厂常规检测指标，包括但不限于：
- 进出水水质：COD、BOD₅、NH₃-N、TN、TP、SS、pH
- 运行参数：DO、MLSS、SRT、HRT、回流比
- 能耗数据：总用电量（月度或年度）
- 污泥量：产泥量、含水率

### 3.4 建模方法
- 机理模型（Process-based）：基于质量守恒、能量守恒
- 数据驱动模型（Data-driven）：随机森林、XGBoost、神经网络
- 混合模型（Hybrid）：机理模型 + 数据驱动校正
