## 二、2026年各月加权平均水质计算

### 2.1 加权平均浓度计算方法

对于每个月 $m$，加权平均浓度 $\bar{C}_m$ 按以下公式计算：

$$\bar{C}_m = \frac{\sum_{i=1}^{46} C_{i,m} \cdot V_{i,m}}{\sum_{i=1}^{46} V_{i,m}}$$

其中 $C_{i,m}$ = 第 $i$ 厂第 $m$ 月的水质浓度，$V_{i,m}$ = 第 $i$ 厂第 $m$ 月处理水量（万m³）。

### 2.2 手工验证方法（Excel操作）

1. 打开XLS → "COD进水"工作表
2. 以1月（2026年）为例：
   - 浓度列：E3:E48（行3至48，共46厂）
   - 水量列：R3:R48（列R为1月水量，按sheet布局顺序第18列起）
   - 手工计算：`=SUMPRODUCT(E3:E48,R3:R48)/SUM(R3:R48)`
   - 预期结果：≈334.0 mg/L
